## Overview

**Single page application**

- Create new tasks
- Mark tasks as done
- Delete tasks

**UI**

- Homemade UI without framework
- Responsive design
- Links in footer open in a new tab

### Running the project

Clone this repository:

```
git clone https://github.com/sebkpf/todo-react.git
cd todo-react
```

Install packages:

```
npm install
```

When installation is complete, run with:

```bash
npm start
```
